使用 C# 实现优先队列

发表于 2020-03-28 分类于 CSharp Valine：

本文主要记录使用 C# 对优先队列的基础实现，其本质是二叉堆，根据比较器不同对应大根堆或小根堆。

做题时遇到了可以使用优先队列的题目，然而发现 .Net 并不提供优先队列，所以就尝试自己手写了一个。

希望可以加深自己对优先队列结构以及堆结构的理解。

字段部分

底层容器为数组，比较器可以使用 Comparison 委托实现，也可以使用 IComparer<T> 接口实现。

从规范的角度来讲，我认为使用 IComparer<T> 接口，将比较器留给泛型类自身进行实现更好，这样在构造优先队列的时候就不需要传入比较器，也可以对泛型进行一定的限制。

当然，对比较器有要求也是可以通过外部传入来自定义比较器的，后面的实际使用部分也会提到如何传入自定义比较器。
